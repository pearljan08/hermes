# Package A

This is package-a, a part of the my-monorepo monorepo. 

## Purpose

Package-a provides a simple function to return a greeting message.

## Installation

To install package-a, run:

```
yarn add package-a
```

## Usage

You can use package-a in your project as follows:

```javascript
import { hello } from 'package-a';

console.log(hello());
```

## Development

To contribute to package-a, clone the repository and run:

```
yarn install
```

Then you can build and test the package using the scripts defined in the package.json.
